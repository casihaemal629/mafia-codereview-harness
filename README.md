## 🔧 Troubleshooting

If the app does not open:

- Check that the download finished
- Make sure you extracted the files if they came in a `.zip`
- Run the app again as a normal user first
- Restart your PC and try again

If Windows blocks the file:

- Right-click the file
- Choose Properties
- Check for an Unblock box near the bottom
- Select Unblock if it appears
- Apply the change and run the app again

If the app opens and then closes:

- Look for a second window behind the first one
- Try opening it from the extracted folder again
- Make sure your antivirus did not remove the file
- Download the latest release again from the release page

If the app is slow:

- Close other apps
- Free up memory
- Use a smaller folder or fewer files for the first run
